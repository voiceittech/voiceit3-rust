A Rust wrapper for VoiceIt's API 3.0 featuring Voice + Face Verification and Identification.

## Installation

Add to your `Cargo.toml`:
```toml
[dependencies]
voiceit3 = { git = "https://github.com/voiceittech/VoiceIt3-Rust.git" }
```

## Getting Started

Sign up for a 10-Day paid trial at [VoiceIt.io](https://voiceit.io/pricing). Visit the settings tab to view your API Key and Token. 

## API calls
You can visit our [HTTP API 3.0 Documentation](https://api.voiceit.io/?rust#introduction) for detailed information on each API call.
